## Именование

Нужно использовать говорящие имена:

Переменная должна своим именем говорить что она содержит

	```js
	const team = {
		name: "Crystal palace",
		id: "t712",
	};
	```

Имя метода должно говорить что он делает

	```js
	const mapOptaTeamsToProjectTeams = (optaTeams) =>
		optaTeams.map((optaTeam) => ({
			name: optaTeam.clubName,
			id: optaTeam.id,
	}));
	```

Если переменная содержит массив/коллекцию, добавляем `s` постфикс

	```js
	const teams = await fetchTeams();
	```

Нет сокращениям.
Лучше выбрать один подход, или сокращать или не сокращать, чтобы было единообразно. Написать полное название длинее, но при этом сразу понятно, что написано и не надо думать.

	Правильно:

	```js
	button
	error
	```

	Неправильно:

	```js
	btn
	e
	```

	Исключения:

	```js
	// event в обработчиках
	const onClick = (e) => {
		e.preventDefault();
	}

	```
